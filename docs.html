What is Complex Mapping?

A complex mapping is a function that takes a complex number and produces another complex number . We write this as .

Complex numbers live in a 2D plane: the horizontal axis is the real part, the vertical is the imaginary part. A complex function can be thought of as a transformation that warps, rotates, stretches, or folds this plane.

For more rigorous mathematical definitions, we recommend reading the mathematical foundations of complex mappings and transformations.

This visualizer lets you draw or plot curves on the z-plane (left), and instantly see how maps those curves onto the w-plane (right) — in real time.

How to Use

1
Enter a mapping function in the top bar (the w = f(z) = input). Type using LaTeX (\sin(z), z^2, \frac{1}{z}) or plain notation (sin(z), z^2, 1/z). Press Apply f(z) or hit Enter.
2
Draw freely on the z-plane (left canvas) with your mouse or touch. The w-plane (right) updates in real time showing the transformed drawing.
3
Add typed equations using the sidebar. Click + y=f(x) to plot a real-valued function (e.g. x^2 draws a parabola), or + z(t) for a parametric complex curve (e.g. e^{it} draws the unit circle). Each equation gets its own color and can be toggled or deleted.
4
Click ⊞ Grid in the toolbar to overlay a coordinate grid on the z-plane and instantly see how the mapping deforms the entire domain — no drawing needed.
5
Use the ƒ Functions panel to browse ready-made formulas organized by category. Click any entry to instantly insert it into the active field and apply it.
6
Open ⚙ Settings to customize the viewport ranges for both planes, the branch cut threshold, grid density, and parametric resolution.

Frequently Asked Questions

What is the w-plane? ▼
The w-plane is the output of the mapping. Every point on the z-plane gets transformed to on the w-plane. It shows you what the function "does" to your drawings and curves.
What is a branch cut? ▼
Functions like log(z) and sqrt(z) are multi-valued — they have more than one valid output per input. The principal branch requires choosing a curve where the function is discontinuous. This visualizer's Branch Cut Threshold (in Settings) controls when a line segment is considered "too long" (a jump) and should be broken instead of connected. Increase the threshold if too many lines are broken; decrease it if you see spurious connecting lines across a branch cut.
What is a conformal map? ▼
An analytic function is conformal (angle-preserving) at any point where its derivative is non-zero. If two curves cross at right angles on the z-plane, their images also cross at right angles on the w-plane. You can verify this: draw two perpendicular lines through a regular point, then watch their mapped images. At critical points (where ), conformality breaks down — angles get multiplied or folded.
How do y = f(x) equations work? ▼
Type any expression in terms of x. The visualizer samples x across the z-plane's real axis range. The result is treated as the y-coordinate (imaginary part), so the plotted point in the z-plane is . For example, x^2 draws a parabola. The result is then mapped through f(z) to the w-plane.
How do z(t) parametric curves work? ▼
Type a complex-valued expression of the real parameter t. The result is a complex number whose real and imaginary parts are the x and y coordinates in the z-plane. Set the t range using the inputs that appear on each parametric equation row. Examples:
  • e^(i*t) for t ∈ [0, 2π] — unit circle
  • t + i*t^2 for t ∈ [−2, 2] — parabola in complex plane
  • cos(t) + i*sin(2t) for t ∈ [0, 2π] — Lissajous figure
Can I use multiple curves at once? ▼
Yes! Use the z-plane sidebar to add as many equations as you like — each gets its own color, visibility toggle, and delete button. You can also draw freehand on top of all typed equations. Everything on the z-plane is simultaneously mapped through f(z) to the w-plane.

My w-plane looks empty or everything is off-screen. Help! ▼
Some mappings (like e^z) can produce values far outside the default [−2, 2] viewport. Open ⚙ Settings and increase the w-plane range (e.g. set Max Re/Im to 10). Alternatively, use the Reset button in settings to return to defaults.
How does this tool compare to GeoGebra or Desmos complex mapping? ▼
While tools like GeoGebra and Desmos are incredibly powerful general-purpose graphing calculators, this application is a specialized complex mapping interactive simulation and complex function calculator. It natively provides a dual-pane setup (z-plane to w-plane), handles branch cuts, and plots complex graph equations in real-time without requiring complex manual setups.
Is this a Conformal map calculator? ▼
Yes! This visualizer serves as a powerful conformal map calculator. By inputting an analytic function, you can observe its conformal properties directly on the Complex plane Grapher 2d, watching how local angles are preserved.
Can I use this as a complex function plotter grapher viewer? ▼
Absolutely. It functions as a complete complex function plotter grapher viewer. Whether you want to explore the visual output of sin(z), plot custom z(t) curves, or just view the mathematical properties of advanced functions, this tool handles it online.
